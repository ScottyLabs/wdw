The Workshops

Start with web development basics, then take any number of topics courses. Finish up with deployment.

Web Development Basics

What is the web? What do I need to learn to make a web site or a web app? What are web sites or web apps anyways?

This course is a rough introduction to web development. It takes about 20 minutes to read, and is a pre-requisite for all the other WDW workshops.

HTML & CSS

This course will teach you everything you need to know to make an informative, unchanging website. Learn how to write HTML, and style it with CSS.
